# WF-05 — AI Sales Report 📊

Relatórios semanais automatizados com insights de inteligência artificial sobre o desempenho de vendas.

## Visão Geral
Este workflow analisa os dados de vendas da semana e gera um relatório detalhado com insights acionáveis, identificando padrões e sugerindo melhorias no funil de vendas.

## Gatilho (Trigger)
- Cron/Schedule: Toda segunda-feira às 08:00.

## Como Usar
1. Importe o `workflow.json`.
2. Configure sua conexão com o CRM.
3. Configure o nó do Claude para análise de dados.
