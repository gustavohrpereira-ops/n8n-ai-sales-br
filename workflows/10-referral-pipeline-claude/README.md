# WF-10 — Referral Pipeline with Claude AI

> Automate your referral program with AI-crafted thank-you messages and partner management.

## Overview

Manages your referral program end-to-end: tracks referrals, sends personalized thank-you messages generated by Claude, calculates commissions, and keeps partners engaged with regular updates.

## Trigger
- **Webhook** — triggered when a new referral is registered
- **Cron** — monthly partner report

## What Claude Does
- Writes personalized thank-you messages for each referral
- Generates a monthly performance summary for partners
- Creates referral program recruitment messages for top clients
- Drafts commission payment notifications
- Suggests ways to reward top referrers

## Output
- Personalized thank-you WhatsApp message to referrer
- Commission calculation and notification
- Monthly partner performance report
- Referral tracking dashboard data

## Required Credentials
- Anthropic API Key (Claude)
- Webhook configuration
- Database/CRM connection
- WhatsApp API

## How to Import
1. Open n8n
2. Click **Import from file**
3. Select `workflow.json` from this folder
4. Configure credentials
5. Activate the workflow

## Why This Matters
Referrals are the highest-converting lead source in consórcio sales. Automating this channel with Claude's personalization capability can dramatically increase referral volume.
