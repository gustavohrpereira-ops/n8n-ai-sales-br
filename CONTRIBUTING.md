<a name="english"></a>
## English Version

Thank you for your interest in contributing! This project aims to democratize AI-powered sales automation for the Brazilian market.

### How to Contribute

#### 1. Suggesting New Workflows
If you have created an n8n workflow that uses Claude 3.5 Sonnet for sales, feel free to open a Pull Request:
- Add the `.json` file to the `/workflows` folder.
- Include a brief explanatory `README.md` within the workflow's subfolder.
- Ensure that the prompts follow the "Structured Output" (JSON) guidelines.

#### 2. Improving Prompts
Our prompt engineering is the heart of the project. If you found a way to make Claude more efficient or precise for the Brazilian market:
- Edit the files in `docs/PROMPT_ENGINEERING.md`.
- Explain the performance gain in the PR description.

#### 3. Reporting Bugs
Open an **Issue** detailing:
- The workflow that presented an error.
- The n8n node where the failure occurred.
- The error returned by the Anthropic API (if any).

### Code & Commit Standards
- Use clear commit messages (e.g., `feat: add follow-up workflow`).
- Keep the documentation updated.

<a name="português"></a>
## Versão em Português

Obrigado por seu interesse em contribuir! Este projeto visa democratizar a automação de vendas com IA para o mercado brasileiro.

### Como Contribuir

#### 1. Sugerindo Novos Workflows
Se você criou um workflow n8n que utiliza o Claude 3.5 Sonnet para vendas, sinta-se à vontade para abrir um Pull Request:
- Adicione o arquivo `.json` na pasta `/workflows`.
- Inclua um breve `README.md` explicativo dentro da subpasta do workflow.
- Certifique-se de que os prompts seguem as diretrizes de "Saída Estruturada" (JSON).

#### 2. Melhorando Prompts
Nossa engenharia de prompts é o coração do projeto. Se você encontrou uma forma de tornar o Claude mais eficiente ou preciso para o mercado brasileiro:
- Edite os arquivos em `docs/PROMPT_ENGINEERING.md`.
- Explique o ganho de performance na descrição do PR.

#### 3. Reporting Bugs
Abra uma **Issue** detalhando:
- O workflow que apresentou o erro.
- O nó do n8n onde a falha ocorreu.
- O erro retornado pela API da Anthropic (se houver).

### Padrões de Código e Commits
- Use mensagens de commit claras (ex: `feat: adiciona workflow de follow-up`).
- Mantenha a documentação atualizada.
