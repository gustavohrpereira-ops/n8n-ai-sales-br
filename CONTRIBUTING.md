# Contributing to n8n-ai-sales-br

Obrigado por querer contribuir! / Thank you for wanting to contribute!

This project builds open-source Claude-powered n8n automation workflows for the Brazilian consórcio market. All skill levels are welcome.

<a name="portugues"></a>
## Como Contribuir (PT-BR)

### Pré-requisitos
- Conta no GitHub
- n8n instalado (local ou cloud) para testar workflows
- Chave de API da Anthropic (Claude) para testar integrações

### Passos

1. **Fork** este repositório
2. Crie uma branch: `git checkout -b feature/minha-contribuicao`
3. Faça suas alterações
4. Teste o workflow no n8n antes de submeter
5. Commit com mensagem descritiva: `git commit -m 'Add: novo workflow de reativação'`
6. Push: `git push origin feature/minha-contribuicao`
7. Abra um **Pull Request** descrevendo o que foi feito

### O que contribuir?

- 🆕 Novos workflows para o ciclo de vendas de consórcio
- 🐛 Correções de bugs nos workflows existentes
- 📝 Melhorias na documentação (PT e EN)
- 🌐 Traduções
- 📸 Screenshots e exemplos de uso
- ✅ Testes e validações

<a name="english"></a>
## How to Contribute (EN)

### Prerequisites
- GitHub account
- n8n installed (local or cloud) to test workflows
- Anthropic API key (Claude) to test integrations

### Steps

1. **Fork** this repository
2. Create a branch: `git checkout -b feature/my-contribution`
3. Make your changes
4. Test the workflow in n8n before submitting
5. Commit with a descriptive message: `git commit -m 'Add: new reactivation workflow'`
6. Push: `git push origin feature/my-contribution`
7. Open a **Pull Request** describing what was done

### What to contribute?

- 🆕 New workflows for the consórcio sales cycle
- 🐛 Bug fixes in existing workflows
- 📝 Documentation improvements (PT and EN)
- 🌐 Translations
- 📸 Screenshots and usage examples
- ✅ Tests and validations

---

## Workflow Standards

All contributed workflows must:

- [ ] Use **Claude AI** as the core intelligence engine
- [ ] Be fully importable into n8n (valid JSON)
- [ ] Include comments explaining each node
- [ ] Have a `README.md` in the workflow folder
- [ ] Follow the naming convention: `XX-workflow-name/workflow.json`
- [ ] Be relevant to the Brazilian consórcio/real estate/sales market

---

## Code of Conduct

- Be respectful and inclusive
- Welcome contributors of all skill levels
- Give constructive feedback on PRs
- Keep discussions focused on the project
