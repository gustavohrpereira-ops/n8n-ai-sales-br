# Contribuindo para o n8n AI Sales BR 🚀

Obrigado por seu interesse em contribuir! Este projeto visa democratizar a automação de vendas com IA para o mercado brasileiro.

## Como Contribuir

### 1. Sugerindo Novos Workflows
Se você criou um workflow n8n que utiliza o Claude 3.5 Sonnet para vendas, sinta-se à vontade para abrir um Pull Request:
- Adicione o arquivo `.json` na pasta `/workflows`.
- Inclua um breve `README.md` explicativo dentro da subpasta do workflow.
- Certifique-se de que os prompts estão em português e seguem as diretrizes de "Saída Estruturada" (JSON).

### 2. Melhorando Prompts
Nossa engenharia de prompts é o coração do projeto. Se você encontrou uma forma de tornar o Claude mais eficiente ou preciso para o mercado brasileiro:
- Edite os arquivos em `/docs/PROMPT_ENGINEERING.md`.
- Explique o ganho de performance na descrição do PR.

### 3. Reportando Bugs
Abra uma **Issue** detalhando:
- O workflow que apresentou erro.
- O nó do n8n onde a falha ocorreu.
- O erro retornado pela API da Anthropic (se houver).

## Padrões de Código e Commits
- Use mensagens de commit claras (ex: `feat: adiciona workflow de follow-up`).
- Mantenha a documentação atualizada.
