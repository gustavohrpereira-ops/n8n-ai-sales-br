# Engenharia de Prompts para Claude 3.5 Sonnet 🧠

Esta pasta contém os prompts otimizados utilizados nos workflows de automação de vendas.

## Estrutura de Prompts

Os prompts são projetados para:
1. **Saída Estruturada:** Sempre gerar JSON para integração direta com n8n.
2. **Persona de Top Broker:** Claude atua como um especialista sênior em consórcio brasileiro.
3. **Análise de Objeções:** Raciocínio multi-etapa para lidar com dúvidas complexas de clientes.

## Como Usar

Copie o conteúdo do prompt desejado e cole no nó do Claude/Anthropic no seu n8n.
