## English Version

### n8n AI Sales BR - Claude-Native Sales Automation 🚀

This repository contains a complete automation framework to accelerate the sales cycle for Brazilian companies, leveraging the power of **Claude 3.5 Sonnet** integrated with **n8n**.

Specifically designed for the Brazilian ecosystem (WhatsApp, CRM Pipedrive/Hubspot, LGPD/GDPR), this system transforms the LLM into a true decision-making and lead qualification machine.

### 🌟 Key Differentiators (Claude for OSS Grant Application)
- **Multi-Step Reasoning:** Claude doesn't just respond; it analyzes objections, scores the lead, and suggests the next action simultaneously.
- **Structured Output:** Prompts optimized to generate deterministic JSON, ensuring n8n never fails during execution.
- **Local Compliance:** Adjusted for Brazilian Central Bank (BACEN) financial regulations and local sales practices.

### 🚀 Large-Scale Impact (10,000+ Users)
This project was born to transform the productivity of **10,000+ financial advisors** in Brazil, allowing repetitive tasks to run on **Autopilot**. 
- **Consultative Focus:** We free up the advisor to focus on human connection and adding real value to the client.
- **Enterprise Ready:** Prepared for scale in major institutions like **Ademicon, Itaú, Porto Seguro, Bradesco, Embracon, HS, Rodobens, Sicoob, Caixa** and investment offices.

### 📂 Repository Structure
- `/workflows`: n8n workflows ready for import.
- `/docs`: Technical documentation, architecture, and impact metrics.
- `/prompts`: Prompt engineering strategies for Claude 3.5 Sonnet.

### ⚡ 10 Available Workflows
| # | Workflow | What it does | Time Saved |
|---|----------|-----------|------------------|
| 01 | Lead Qualification | Classifies HOT/WARM/COLD leads with Claude | ~3h/day |
| 02 | WhatsApp Follow-up | Automatic and personalized follow-ups | ~2h/day |
| 03 | Proposal Generator | Automatically generates HTML/PDF proposals | ~1h/lead |
| 04 | CRM Enrichment | Enriches lead data via AI + Company ID (CNPJ) | ~4h/week |
| 05 | AI Sales Report | Weekly reports with AI insights | ~3h/week |
| 06 | Client Onboarding | Personalized welcome for new clients | ~30min/client |
| 07 | Cold Lead Reactivation | Reactivates cold leads (30-180 days) | ~2h/week |
| 08 | Objection Handler FAQ | Responds to objections and FAQs with AI | ~1h/day |
| 09 | Contemplation Notification | Notifies and guides contemplated clients | ~45min/event |
| 10 | Referral Pipeline | Automates referral programs | ~3h/week |

### 🛠️ How to Install
1. Clone this repository.
2. Import JSON files from `/workflows` into your n8n instance.
3. Configure environment variables with your Anthropic and Evolution API keys.
